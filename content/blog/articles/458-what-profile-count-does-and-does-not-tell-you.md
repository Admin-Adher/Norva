# What Profile Count Does and Does Not Tell You

> **In short:** Profile count tells you how many separate viewing contexts a current plan can provide. Those contexts can organise progress, history, favourites, and preferences. The number does not, by itself, state how many devices may be used, how many sessions may play together, whether profiles are private, whether parental controls exist, or whether media access is included.

Confusing capacity with permission creates household rules that the product never made. Read profile count narrowly and verify every other condition separately.

## What profile count describes

A profile is a persistent viewing context. When supported, it can separate:

- playback progress;
- history;
- favourites;
- audio and subtitle preferences;
- source-derived recommendation context.

Current profile allowances and conditions belong on Norva's official pricing page. Recheck them instead of embedding an old number into household policy.

## It does not describe device limits

One person may use the same profile across supported devices because Norva can preserve account state across screens. That does not convert profile count into a number of authorised devices.

Device support requires its own current product documentation.

## It does not prove simultaneous playback

Two profiles do not automatically mean two concurrent sessions. Current plan terms, source conditions, rights, and device behaviour govern the intended use.

Never write a household rule that multiplies profiles by sessions without a verified source.

## It is not account security

A profile label is not a password. Anyone who can access the signed-in account may be able to see or switch contexts. Device locks, account authentication, and sign-out practices belong to other layers.

Norva's public privacy and terms pages explain current account and data conditions.

## It is not parental control

Profile capacity does not establish content filtering, classification, time limits, or protected profile access. Do not market Norva Family or any profile count as a parental-control system without verified product documentation.

## It does not include media access

Norva is software for organising and playing a compatible media source the user owns or is authorised to use. The subscription does not include a media catalogue.

Profile count cannot create source rights, tracks, or availability.

## Use count for one decision only

Ask how many persistent contexts need separate progress, history, favourites, or preferences. The [profile-count worksheet](/blog/how-many-household-profiles-do-you-actually-need/) turns that need into an estimate.

Then compare the estimate with current plan capacity. Evaluate devices, concurrency, privacy, and rights independently.

The [separate-versus-shared decision framework](/blog/separate-profiles-or-one-shared-profile-a-decision-framework/) helps determine context demand. The [complete profile guide](/blog/the-complete-guide-to-household-media-profiles/) covers naming and lifecycle.

Choose a plan only after reviewing current prices, conditions, profile needs, and the broader product fit. Avoid evergreen articles with stale quoted prices.

## Common misconceptions and limitations

A higher profile count is not universally better. Unused profiles can create selection clutter, while insufficient contexts can mix state. The right count depends on recurring viewing patterns.

This explainer does not interpret contractual rights beyond the public terms and should not replace current product or legal review.

## Pair the count with one verified requirement

When comparing a household need with a plan, write the required number of distinct profile contexts beside the current official profile capacity. Keep simultaneous playback, device support, source access, and security in separate rows so the count is not reused as evidence for unrelated limits.

## Frequently asked questions

### Does one profile equal one person?

Not always. A recurring group may own one shared context, while personal profiles can serve one person across supported devices.

### Can two profiles use the same device?

Profiles are designed as contexts within an account when available. Confirm the active label before playback.

### Does Norva Family mean parental controls?

Do not infer that from the name or profile capacity. Use only verified feature documentation.
