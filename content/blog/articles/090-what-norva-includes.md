# What Norva Includes—and What You Still Need to Provide

> **In short:** Norva includes software for interface, organisation, playback, and synchronisation across supported web, mobile, and TV devices. It does not include a media catalogue or media access. You provide a compatible source you own or are legally authorised to use, the associated rights, supported devices, connectivity, and local storage when needed.

This division matters before subscribing. It prevents a software feature list from being mistaken for the media, hardware, network, or permissions required to use it.

## Included in the Norva software experience

Current Norva product information describes:

- a unified interface for a connected compatible source;
- catalogue browsing, organisation, and search;
- grouping of media variants;
- playback on supported devices;
- synchronised progress, history, favourites, and preferences;
- source-based recommendations;
- retained language and subtitle preferences when those tracks exist;
- web access in a compatible browser without installation;
- a remote-oriented TV experience;
- conditional offline access on supported mobile workflows;
- profile capacity according to the chosen plan.

These are software capabilities. Their usefulness still depends on the actual source data and media options available.

## Not included with the subscription

Norva does not include a media catalogue or media access. It does not create a track, title, category, guide entry, or version missing from the connected source.

It also should not be assumed to provide:

- every device or source compatibility;
- a guaranteed resolution or audio format;
- unrestricted offline availability;
- a particular number of simultaneous streams or devices;
- universal network performance;
- rights to media supplied elsewhere.

For the core product explanation, read [what Norva is](/blog/what-is-norva-media-player/).

## What the user provides

You remain responsible for:

1. **The source:** a compatible media source you own or are legally authorised to use.
2. **The rights:** permission for the intended access, playback, and any offline use.
3. **The account details:** accurate sign-in information and secure recovery methods.
4. **The devices:** supported hardware, operating systems, browsers, and input methods.
5. **Connectivity:** a suitable network for setup, connected playback, and sync.
6. **Storage:** enough device space for eligible offline items and normal operation.
7. **Source quality:** accurate metadata, available versions, and tracks exposed by that source.

The [compatible-source setup guide](/blog/connect-compatible-media-source-norva/) helps prepare these inputs without assuming a particular provider.

## How the plans differ

The Norva plan includes up to two profiles. Norva Family includes up to five. Profile capacity helps households keep progress and preferences separate.

Do not infer device or simultaneous-playback limits from those profile counts. Consult the current official pricing and terms on the day you decide. The [Norva plans guide](/blog/norva-plans-explained/) uses profile needs and current terms rather than hard-coded price claims.

## Conditional features need a real test

Some capabilities depend on inputs outside the software. For example:

- language choices depend on the source version and media tracks;
- guide information depends on source data;
- recommendations are drawn from the connected source;
- offline access depends on device, source, media, and rights;
- cross-device updates depend on the correct account, profile, and connectivity.

Use representative items during evaluation. The [Norva trial checklist](/blog/norva-trial-checklist/) records pass, recheck, unavailable, and not-applicable outcomes.

## Common mistakes and limitations

- Assuming software subscription and media access are one product.
- Treating a source login as proof of legal authorisation.
- Inferring device limits from profile counts.
- Expecting Norva to correct missing source metadata automatically.
- Assuming every media item has the same audio and subtitle options.
- Treating offline availability as universal.
- Relying on an old price or feature screenshot.

Product terms, prices, and compatibility can change. Recheck official pages immediately before publication or purchase.

## Frequently asked questions

### Does Norva include a media library?

The software organises the catalogue exposed by your connected compatible source. The subscription does not include a media catalogue or media access.

### Do I need an internet connection?

A connection is needed for setup, connected source access, and synchronisation. Eligible offline playback can work without a live connection after preparation, subject to device, source, media, and rights.

### What should I verify first?

Confirm that you have a compatible source you are authorised to use and supported devices for the workflows you care about. Then test representative items.
