# Mobile, Web, or TV: Which Norva Experience Fits Each Moment?

> **In short:** Choose mobile for a close, personal screen, Web for quick access in a compatible browser without installation, and TV for a sofa-based interface designed around a remote. Norva is available across all three, and the same account can retain catalogue state, progress, favourites, history, and preferences on supported devices.

The right screen is contextual. A television can be ideal for a shared evening but awkward for entering text. A browser can be immediate on a computer but unsuitable when you want a remote-first experience. Mobile can support a private, close interaction without replacing the room-scale role of TV.

## Compare the interaction, not just screen size

| Context | Mobile | Web | TV |
| --- | --- | --- | --- |
| Typical distance | Handheld or nearby | Desk or laptop distance | Sofa or room distance |
| Primary input | Touch or device controls | Keyboard, pointer, or compatible controls | Remote and D-pad |
| Installation | Application required | No installation in a compatible browser | TV application on a supported device |
| Strong fit | Personal access and handoff | Quick browser access | Shared, remote-first viewing |
| Main caution | Small screen and interruptions | Browser and computer context | Text entry and distant readability |

This is a context matrix, not a performance comparison. Actual availability depends on supported devices, a compatible browser where relevant, and the connected source.

## Choose mobile for close, personal interaction

Mobile can fit:

- selecting a title in a personal setting;
- starting a session before moving to another supported screen;
- checking favourites or progress close at hand;
- using a touch-oriented interface.

Do not assume that every media item can be used offline. Offline access exists only when the device, source, and associated rights permit it.

For a structured handoff, use [start on mobile, finish on TV](/blog/start-mobile-finish-tv/).

## Choose Web for immediate browser access

Norva's Web experience opens in a compatible browser without installation. It can be useful when:

- you are on a supported computer where installing an app is undesirable;
- keyboard and pointer input make search convenient;
- you want a larger personal display;
- you need to access the same account from the browser context.

Web convenience does not make every computer appropriate. On a shared or public machine, consider account privacy, sign out when finished, and avoid saving credentials through a browser you do not control.

Read [when a browser-based media player is convenient](/blog/when-to-use-browser-media-player/) for a fuller decision.

## Choose TV for remote-first, room-scale use

Norva's TV interface is designed for remote-control navigation. TV fits:

- a shared room;
- a longer viewing session;
- browsing from the sofa;
- moving through rows and details with a D-pad.

The interface must remain readable at distance and make focus obvious. If TV navigation feels slow, review [how to navigate with a remote](/blog/navigate-media-app-tv-remote/) rather than treating it like a touch screen.

## Use continuity instead of choosing permanently

The three experiences are complementary. A practical sequence can be:

1. save a title to Favorites on mobile;
2. inspect it later in a compatible browser;
3. play or resume it on TV;
4. continue on another supported screen when needed.

Norva states that a single account preserves catalogue, progress, favourites, history, and preferences across supported devices. The source still supplies the media and determines available language, subtitle, and offline options.

The [playback progress sync guide](/blog/playback-progress-sync-explained/) explains the account, profile, media identity, and connectivity checks behind a reliable handoff.

## Limits to verify before deciding

- Supported device and browser requirements.
- Whether the connected source is available on the chosen screen.
- Language and subtitle tracks for the specific media.
- Offline eligibility for the device, source, and associated rights.
- Current plan terms and profile capacity.

Norva includes up to two profiles; Norva Family includes up to five. Profile capacity should not be interpreted as a stated device or simultaneous-playback limit.

## Frequently asked questions

### Is one Norva experience more complete than the others?

The public product information confirms availability on Web, mobile, and TV and identifies platform-specific interaction differences. Check the current supported-device information and live interface for any feature-specific requirement rather than assuming complete parity.

### Can I begin on one screen and continue on another?

Norva synchronises playback progress across supported devices. Use the same account, profile, title or episode, and intended version, and confirm connectivity.

### Does Web require installation?

No. Norva states that its Web experience works in a compatible browser without installation.
