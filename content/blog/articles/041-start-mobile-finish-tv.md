# Start on Mobile, Finish on TV: A Practical Cross-Screen Workflow

> **In short:** Use the same Norva account and profile on both supported devices, stop normally on mobile, and open the same title, episode, and version on TV. Confirm the offered resume point before selecting it. If it is wrong, check profile, version, recent playback, and connectivity one at a time.

A cross-screen handoff is most useful when it feels uneventful. Mobile suits a short session or private browsing; a TV suits a shared, sofa-based session. The transition fails when “the same title” hides a different profile, episode, or version.

## Prepare both screens once

Before relying on handoff:

- sign in to the same Norva account on supported mobile and TV devices;
- select the intended profile on both;
- confirm that the compatible media source is available;
- make sure the TV can be navigated comfortably with its remote;
- review the language and subtitle preferences associated with the profile.

Norva synchronises catalogue state, playback progress, history, favourites, and preferences across supported devices. Media availability still depends on the compatible source you are authorised to use.

For the underlying model, read [how playback progress sync works](/blog/playback-progress-sync-explained/).

## Start the session on mobile

Open the title details before playback and verify:

1. exact title or series;
2. season and episode where relevant;
3. selected version;
4. audio and subtitle options actually available;
5. the active profile.

Begin playback. When you are ready to move, stop or exit normally rather than force-closing the app during an active transition. Return to the title details or library and give the connected device an opportunity to reflect the new position.

This workflow does not promise a fixed sync delay. Observe the state shown by the current product and connection.

## Resume from the TV

On TV:

1. open Norva with the remote;
2. confirm the same profile;
3. look for the title in Continue Watching or search for it;
4. open the same version;
5. inspect the displayed progress;
6. choose Resume only when the position looks plausible.

If the TV interface is unfamiliar, first review [efficient remote navigation](/blog/navigate-media-app-tv-remote/). Guessing with repeated directional presses can move focus away from the intended card.

The card is useful when a mismatch occurs because it distinguishes identity from time. It contains no password, source credential, or payment data.

## If the TV offers the wrong position

Work through these checks in order:

- Is the TV on another profile?
- Did mobile and TV open different versions?
- Did another supported screen play the title afterward?
- Was the mobile device offline when the session ended?
- Is the title currently available through the same source record?
- Does the mobile device still show the expected position?

Change one variable, then retry. Repeatedly scrubbing on both devices creates new progress events and makes the original problem harder to describe.

Use [the full progress-sync troubleshooting guide](/blog/playback-progress-not-syncing/) if the same mismatch survives a controlled retry.

## Preserve language choices during the move

Norva can retain audio and subtitle preferences across supported devices when those options are available. The TV version must still provide the corresponding track.

Before resuming:

- confirm that the same version is selected;
- inspect the available tracks;
- apply the preferred language if it was not selected;
- avoid assuming every episode has identical options.

For a dedicated workflow, see [how language preferences follow between devices](/blog/sync-audio-subtitle-preferences/).

## Common mistakes and limitations

- Switching profiles without noticing.
- Resuming a different episode or version.
- Expecting a disconnected device to show the newest state immediately.
- Using Favorites as evidence of playback progress.
- Scrubbing on both screens while diagnosing.
- Assuming every source, item, and supported device exposes identical options.

## Frequently asked questions

### Do I need to search for the title again on TV?

It may appear in Continue Watching when progress is recorded. If it does not, search or browse to the same title and verify the profile and version before diagnosing sync.

### Can audio and subtitle choices carry over too?

Norva can retain those preferences across supported devices, but the corresponding options must exist for the selected source version and media item.

### What if the mobile device ended the session offline?

The newest state may not be available to another screen until connectivity returns. Reconnect, verify the mobile position, then inspect the TV again.
