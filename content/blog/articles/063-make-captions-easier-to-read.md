# How to Make Captions Easier to Read on Phones and TVs

> **In short:** Test captions at the distance and in the room where you actually watch. Check text size, contrast, background treatment, line length, position, and whether the text remains readable over bright, dark, and busy scenes. Use device or application caption controls when available, but remember that styling options differ by platform and source.

Readable captions are not simply “large text.” A huge line with weak contrast can be harder to follow than a moderately sized line with a clear background. The right setup also differs between a phone held nearby and a television viewed across a room.

## Begin with the actual caption track

Confirm that the selected text track serves the viewer’s need. The W3C explains that captions include dialogue and important non-speech audio. A generic subtitle track may not include the same information.

If text is absent rather than hard to read, use the [missing-subtitles checklist](https://norva.tv/blog/missing-subtitles-checklist/). If you are selecting a track for the first time, follow [the low-interruption subtitle routine](https://norva.tv/blog/select-video-subtitles/).

## Run the three-scene readability check

This is the original evidence exercise for the article. Choose one passage with captions, then observe text over:

1. a bright background;
2. a dark background;
3. a visually busy or changing background.

Use the normal viewing distance, posture, room lighting, and eyewear. For each scene, record:

| Question | Bright | Dark | Busy |
| --- | --- | --- | --- |
| Can I identify every letter quickly? |  |  |  |
| Does text separate from the picture? |  |  |  |
| Are lines short enough to scan? |  |  |  |
| Does the position avoid important visuals? |  |  |  |

This is a comfort check, not a formal contrast measurement. W3C contrast guidance explains why foreground and background differentiation matters, but a viewer observation does not certify a numerical ratio.

## Adjust size without sacrificing scanning

Increase size until text is legible from the real viewing position. Then check whether lines wrap excessively or cover too much of the picture. On a phone, an overly large setting can create many short lines. On TV, a setting that looks adequate while standing near the screen may be too small from the sofa.

Change one setting at a time and replay the same three scenes. If controls are provided by both the operating system and the application, document which layer you changed so you can reverse it.

## Improve separation from the picture

Where supported, a background box, shadow, or outline can help letters remain distinct across changing scenes. Prefer a treatment that improves readability without hiding more of the image than necessary.

Do not assume one colour works everywhere. Recheck bright, dark, and busy scenes. If the selected source track contains its own styling, device-level choices may have limited effect.

## Check room and screen conditions

On TV:

- sit in the usual seat;
- reduce glare where practical;
- check that display processing has not made text edges uncomfortable;
- make sure the interface itself is readable from the sofa.

On phone:

- hold the device at a comfortable distance;
- avoid blocking captions with controls or fingers;
- use a stable orientation that does not force constant refocusing;
- consider room brightness and screen reflections.

The article on [TV-app readability from the sofa](https://norva.tv/blog/tv-app-sofa-readability/) covers the wider interface beyond captions.

## Review timing and placement separately

Perfect styling cannot fix text that appears at the wrong time. Use a dialogue-heavy passage to check whether the caption remains close enough to the spoken line to follow. Also verify that placement does not consistently cover names, translations, or other important on-screen information.

If timing is incorrect only for one item or track, record the item, version, track label, and timestamp. Availability and quality depend on the media and authorised source.

## Common mistakes and limitations

- Adjusting text while standing close to a TV rather than from the real seat.
- Testing only one dark scene.
- Enlarging text until line wrapping harms reading flow.
- Assuming subtitles and captions always contain equivalent information.
- Claiming formal accessibility compliance from a visual check.
- Expecting the same styling controls on every device or source.

## Frequently asked questions

### What matters more: caption size or contrast?

Both matter, along with background treatment, line length, position, timing, distance, and room conditions. Improve the weakest factor and retest the same scenes.

### Why are captions readable on my phone but not my TV?

Viewing distance, display size, room glare, styling controls, and the platform’s rendering can differ. Configure and test each screen in its real context.

### Does a background box always improve captions?

It often improves separation, but it can cover more of the picture. Use it when available, adjust conservatively, and check multiple scene types.

### Can this checklist prove WCAG compliance?

No. It is a viewer comfort check. Formal conformance requires evaluation against the applicable criteria and measurement methods.
