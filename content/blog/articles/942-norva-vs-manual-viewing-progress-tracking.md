# Norva vs. Manual Viewing-Progress Tracking

> **In short:** Norva can keep playback progress across supported devices under the same account, while a manual note or spreadsheet requires deliberate updates. Manual tracking may remain adequate for one viewer, one screen, and a small list. Compare actual capture effort, retrieval, correction, profile separation, privacy, resilience, and current cost over several real sessions before deciding.

This comparison is about workflows, not which method sounds more modern. A manual system can be transparent and portable, while an integrated system can reduce repeated entry. Both can fail if the item identity, profile, or source version is unclear.

## Define the manual baseline

Choose the manual method you would genuinely maintain: paper, a local note, a spreadsheet, or a task list. Define the fields before testing, such as title shorthand, season, episode, elapsed position, date, and viewer context.

Do not store source credentials, full private titles, or unnecessary household identities. A manual log becomes another dataset that needs access control, retention, and backup decisions.

## Define the Norva baseline

Use a compatible source you own or are legally authorized to use, one Norva account, the intended profile, and supported screens. Norva provides organization, playback, and supported synchronization; it does not provide the media catalog.

Complete the [Norva evaluation framework](/blog/norva-evaluation-framework/) so the source, screen, and profile prerequisites are consistent across tests.

## Measure capture effort

For each viewing session, count deliberate progress actions. Manual tracking may require opening a separate tool, identifying the item, entering a position, and saving. Norva may update progress through the supported playback workflow, but the result still needs verification under the test conditions.

Record time and actions immediately after the session. Memory-based estimates tend to favor the method that felt easier.

## Measure retrieval and continuation

At the next session, time how long it takes to find the item and reach the intended position. For manual tracking, include reading the record and seeking. For Norva, record the displayed resume point and the actual playback position separately.

The [cross-screen continuity guide](/blog/evaluate-norva-cross-screen-continuity/) provides a controlled A-to-B-to-A trace when two screens are essential.

## Test corrections

Create one benign correction case: stop at the wrong episode, record an inaccurate manual position, or intentionally resume and exit at a new point. Measure how clearly each workflow exposes the error and how much effort is needed to correct it.

An automated state that is difficult to understand can be more frustrating than a manual note. A manual system that is easy to edit but often forgotten can be equally unreliable.

## Compare household separation

If more than one person watches the same work, test whether each method prevents progress collisions. A manual system needs a viewer field or separate logs. Norva uses profile context where supported, subject to current plan limits.

Do not average two viewers' positions into one record. Score clarity and selection effort for each household member.

## Compare privacy and resilience

Map who can see progress on a shared screen and who can open the manual log. Review backups, exports, device loss, accidental deletion, and support evidence. More control is not automatically more private if the manual file is broadly shared.

Norva's privacy disclosures and the manual tool's privacy model should be reviewed separately. Avoid assuming that local storage is always private or that cloud synchronization is always available.

## Compare total cost

Use current Norva pricing and terms rather than a fixed number. For the manual workflow, include setup, repeated entry, maintenance, backup, correction, and household training. Time has value, but assign a rate only if it reflects the evaluator's real decision.

Use the [trial evaluation scorecard](/blog/norva-trial-evaluation-scorecard/) to decide before any trial or billing deadline.

## Frequently asked questions

### Is manual tracking always free?

Not necessarily. The tool may be free, but setup, entry, backup, correction, and maintenance consume time and may involve another service.

### Does Norva guarantee progress for every source and device?

No universal guarantee should be inferred. Test the exact compatible source, media, profile, and supported screens that matter to the household.

### When is a manual log enough?

It may be enough for a small, stable list, one viewer, and one primary screen when the person reliably updates it and accepts the privacy and recovery model.
