# How to Add a Second TV Without Losing Your Preferences

> **In short:** On a supported second TV, follow Norva's current official setup or sign-in flow, use the same account, and select the intended profile. Then verify the compatible source, favourites, progress, and language preferences. Do not interpret profile capacity as a published device or simultaneous-playback limit.

Adding another screen should extend the same account experience, not create a second catalogue by accident. Most problems begin with an unintended profile, a different source record, or assumptions about a setup screen that has changed.

## Capture the first TV's context

Before touching the second TV, note:

- Norva account email, without recording the password;
- active profile;
- one favourite title;
- one title with an unmistakable resume point;
- preferred audio and subtitle language;
- the compatible source currently connected;
- whether the first TV appears in any current device controls.

This creates a safe comparison. It does not require exporting personal data or taking screenshots containing credentials.

## Use the current official setup flow

Install or open Norva only through the supported route for that television. Follow the live instructions to sign in or connect the TV. Do not rely on guessed menu labels from an older article.

If the current setup presents a pairing route, follow the on-screen instructions and use the [TV account pairing guide](/blog/pair-tv-with-norva-account/) as a companion checklist. If the interface instead asks you to sign in directly, use that live route.

Never share a pairing code publicly, and verify the official address before approving a request on another device.

## Select the intended profile

Norva includes up to two profiles, while Norva Family includes up to five. Those numbers describe profile capacity. They must not be converted into an unsupported statement about devices or simultaneous playback.

Choose the same profile if you want its existing progress, favourites, and preferences. Choose another existing profile when the second TV belongs to a household member who needs separate state.

Review current plan information on the [Norva pricing page](https://norva.tv/#pricing) rather than placing a changing price or condition in this guide.

## If preferences appear missing

Check in this order:

1. Confirm account and profile.
2. Confirm the second TV can connect.
3. Open the same title and version.
4. Inspect the language and subtitle options supplied for that media.
5. Compare one favourite and one resume point.
6. Avoid changing several preferences at once.

Norva retains progress, history, favourites, and preferences across supported devices. Language options still depend on the source and media. The [playback progress model](/blog/playback-progress-sync-explained/) explains how identity and connectivity affect comparison.

## Review device access afterward

Once the TV works, inspect any available account or device controls. Remove old or unfamiliar devices only after confirming their identity.

Use [the trusted-device review checklist](/blog/review-trusted-device-list/) to separate a renamed television from an unknown session. Never expose source credentials or passwords in a support request.

## If the second TV cannot connect

Collect:

- TV platform or model information;
- Norva app version if visible;
- the setup stage where the issue occurs;
- exact on-screen error wording;
- whether the official support page is reachable on another device;
- whether the same account still works on the first TV.

Then contact [Norva Support](https://norva.tv/support). Do not reset the first working TV merely to make both screens identical unless support specifically recommends it.

## Common mistakes and limitations

- Creating a new account instead of using the existing one.
- Selecting the wrong profile.
- Assuming profile count equals device count.
- Expecting a missing source track to appear through preference sync.
- Using an old pairing address or expired code.
- Removing the first TV before the second is verified.
- Claiming a universal device or simultaneous-use limit without current public evidence.

## Frequently asked questions

### Will my favourites and progress appear on the second TV?

Norva synchronises favourites, progress, history, and preferences across supported devices. Verify the same account, profile, source record, and connectivity.

### Do two profiles mean only two TVs?

No such conclusion should be drawn. Public plan information describes profile capacity, not a verified device or simultaneous-playback limit.

### Should I remove the first TV?

Not simply because a second TV is added. Review device controls and current plan terms, then remove only devices that are old, transferred, or unfamiliar.
