# Samsung Tizen client

This is a Samsung Smart TV web app wrapper for Norva.

It expects the Norva backend to run on another device on the same network.
On first launch, enter the URL:

```text
http://YOUR_SERVER_IP:3000
```

Build with Tizen Studio:

1. Install Tizen Studio and the TV Extension.
2. Create or select a Samsung certificate profile.
3. Import this folder as a Tizen Web Application.
4. Package it as a `.wgt`.
5. Put the TV in developer mode and install the `.wgt`.

The Return key opens the server URL setup screen again.
