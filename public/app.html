    Content & Discovery

    Catalog Region Organizes Live TV, logos, categories and recommendations. It never blocks content access.
    Group Duplicate Movies & Series Show one card per title even when several versions exist across sources or categories. The best version is picked automatically when playing.
    Preferred Audio Language Used to rank versions and pick the best default. Unknown language data stays visible.
    Preferred Subtitle Language Used as a soft signal for VOST/subtitled variants, not as a hard filter by default.
    Strict Language Matching Only demotes versions confirmed incompatible. Unknown language data is kept to avoid hiding good content.
    Favorite Genres Used as a soft discovery signal for Home, Movies and Series. Unknown genre data stays visible.
    Preferred Version Quality Quality to auto-pick when several versions exist.
    TMDB API Key Optional. Enables real genres, duration filter, better posters and smarter duplicate matching. Free key at themoviedb.org → Settings → API.
    Metadata Enrichment Runs automatically after each sync when a TMDB key is set.
    Restore incorrectly hidden titles Some movies/series may have been hidden by "Hide broken" because of a temporary connection refusal (401/403). Click to restore them — they were never actually broken.

    Encoder Settings

    Hardware Encoder Select which GPU encoder to use for transcoding. Auto will use the best available.
    Max Resolution Limit transcoded output resolution. Lower = faster, more compatible.
    Quality Preset Higher quality uses more bandwidth and CPU/GPU.
    Audio Mix Preset How multi-channel audio (5.1/7.1) is mixed down to stereo for browser playback.

    Upscaling

    Enable Upscaling Upscale lower-resolution content to a higher resolution during transcoding.
    Upscale Method Hardware is faster; Software (Lanczos) is higher quality.
    Upscale Target Target resolution for upscaled output.

    Stream Processing

    Auto Transcode (Smart) Automatically detect stream codecs and only transcode/remux when needed. Adds ~1-3s probe delay on first play.
    Force Audio Transcode Transcode audio to AAC for browser compatibility. Enable if you get video but no audio (fixes Dolby/AC3/EAC3).
    Force Video Transcode Force full video transcoding (burn-in subtitles, incompatible video). Uses HW encoder.
    Force Remux Remux streams to MP4 container. Enable for raw .ts streams from IPTV middleware like m3u-editor or dispatcharr.
    Stream Output Format Container format for Xtream streams. Try TS if you experience buffering issues.

    HTTP Settings

    User-Agent HTTP User-Agent for stream requests. Try VLC if your provider blocks Chrome.
    Custom User-Agent Enter your own User-Agent string.

    Network

    Force Backend Proxy Route all streams through the server to bypass CORS restrictions. Enable if streams don't play directly.
